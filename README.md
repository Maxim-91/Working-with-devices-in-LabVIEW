### LabVIEW is a development environment and platform for running programs written in the National Instruments "G" graphical programming language.

#### Uni-t 70B, Uni-t 70D multimeters were used in the work. They are devices of a wide profile and are used to measure various physical quantities in a wide range. These devices have the ability to transmit the values ​​of measured values ​​to a PC using an optical RS-232 interface.
